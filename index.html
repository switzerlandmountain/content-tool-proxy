Content Tool API Proxy

This is a GitHub Pages proxy for the Content Tool. It helps resolve CORS issues and provides a stable endpoint for API interactions.

Available Endpoints

  • Status Endpoint - Check if the proxy is working
  • Outline Endpoint - Generate content outlines

Configuration

To use this proxy with your Content Tool, configure the webhook URL as:

https://[your-username].github.io/content-tool-proxy/api/outline
