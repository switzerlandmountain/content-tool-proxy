Content Tool API Proxy

Status: Active

This is a static proxy for the Content Tool API. It helps bypass CORS restrictions when making requests from browser applications.

Available Endpoints

  • /api/outline - Generate content outlines
  • /api/status - Check API status

How to Use

Send a POST request to /api/outline with your outline request data:

fetch('https://your-username.github.io/content-tool-proxy/api/outline', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    eventId: "evt_123456789",
    title: "Cloud Security Best Practices",
    mainKeyword: "cloud security",
    secondaryKeywords: ["cybersecurity", "data protection"]
  })
})
.then(response => response.json())
.then(data => console.log(data));

How It Works

This proxy uses GitHub Actions to process API requests. When you send a request to one of the endpoints:

  1. The request is forwarded to a GitHub Action workflow
  2. The workflow processes your request using OpenAI's API
  3. Results are returned to you without CORS issues

All requests are processed securely and no data is stored permanently.
